# TaskFlow - Gestionnaire de tâches d'équipe

Projet pratique évalué d'Architecture Front-End dans le cadre de l'Unité de Spécialité "IDE & Frameworks" à l'École Nationale Supérieure Polytechnique de Maroua.

## Fonctionnalités implémentées
- Configuration d'architecture moderne avec Vite.js.
- Modélisation de composants atomiques et gestion saine de l'état (Immuabilité).
- Formulaires contrôlés avec levée d'état (Lifting State Up).
- Persistance synchrone dans le LocalStorage (via un hook personnalisé `useLocalStorage` pour le bonus).
- Navigation interne fluide de type Single Page Application (SPA) avec `react-router-dom`.
